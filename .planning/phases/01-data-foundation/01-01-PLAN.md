**validate_holdout.py:**
- Load `research/data/holdout_realworld.jsonl`
- Assert file exists (exit 1 with "Holdout file not found" if missing)
- Assert line count >= 200 and <= 500
- Assert every line parses as valid JSON with keys: `text`, `label`, `vector`, `source`
- Assert `label` values are only `"scam"` or `"safe"`
- Assert `source` values are only `"ftc"`, `"reddit_rscams"`, `"phishtank"`, `"huggingface_phishing"`, `"huggingface_difraud"`, `"huggingface_sms_spam"`, or `"manual"`
- Assert at least 3 unique `source` values present (per D-01: all three source families)
- Assert at least 40 samples have `label == "safe"` (per D-04: ~50 safe messages)
- Print summary: total count, per-label count, per-source count, per-vector count

**validate_synthetic.py:**
- Accept `--check-counts`, `--check-jsd`, `--check-negatives` flags (run all if no flag)
- `--check-counts`: Load `research/data/synthetic_scam_v1.jsonl`, assert total lines >= 16000, assert all 8 vectors present: `crypto_investment`, `romance_grooming`, `tech_support`, `government_impersonation`, `lottery_reward`, `urgency_payment`, `phishing`, `remote_access`; assert `safe` label count is between 40% and 60% of total (per D-13: 50:50 ratio)
- `--check-jsd`: Compute token-unigram Jensen-Shannon divergence between all scam vector pairs using `scipy.spatial.distance.jensenshannon`. Assert no pair has JSD < 0.05 (mode collapse threshold per D-08). Print full divergence matrix.
- `--check-negatives`: Assert safe-class samples include at least 4 hard negative types from D-09: bank alerts, delivery notifications, 2FA codes, medical alerts. Check by searching for keywords: `["bank", "fraud alert"]`, `["delivery", "package", "tracking"]`, `["verification code", "2FA", "one-time"]`, `["appointment", "prescription", "pharmacy"]`

**validate_quality.py:**
- Load `research/data/synthetic_scam_v1.jsonl`
- Assert zero exact-duplicate `text` values (after stripping whitespace)
- Assert zero samples with `text` split by whitespace having fewer than 15 tokens
- Print: total samples, duplicates found (should be 0), sub-15-token samples found (should be 0)

**validate_split.py:**
- Load `research/data/synthetic_scam_v1.jsonl` and `research/data/test_split.jsonl`
- Assert both files exist
- Compute total = synthetic + test; assert train+val is ~80% of implied total (tolerance: 78-82%), test is ~10% (tolerance: 8-12%)
- Assert zero text overlap between synthetic_scam_v1.jsonl and test_split.jsonl (no contamination)
- Assert all 8 vectors + safe class appear in both files (stratification check)
- Print: per-file counts, per-vector distribution in each file
