  <action>
Create `research/scripts/filter_and_split.py` that implements the full quality pipeline. The script runs sequentially through 4 stages, halting on any gate failure.

**Stage 1: JS Divergence Gate (D-08, mandatory pre-filter)**

- Load `research/data/synthetic_raw.jsonl`
- Compute token-unigram Jensen-Shannon divergence for all scam vector pairs using `scipy.spatial.distance.jensenshannon`
- Implementation: for each vector, collect all texts, lowercase, split on whitespace to get tokens, build frequency distribution. Normalize to probability. Compute JSD for every pair of vectors.
- Gate: if ANY pair has JSD < 0.05, halt with error message naming the collapsed pair. Do NOT proceed to filtering.
- On pass: log full divergence matrix to stdout and save to `research/data/jsd_matrix.json`

**Stage 2: LLM Self-Consistency Check (Pass 1 of two-pass filter)**

- Use `transformers.pipeline("zero-shot-classification", model="facebook/bart-large-mnli", device=-1)` (CPU)
- For each sample, run zero-shot classification with candidate_labels `["scam", "safe"]`
- Keep sample if predicted label matches `sample["label"]` AND confidence >= 0.6 (starting threshold per RESEARCH.md open question 2)
- Log discard statistics per vector. If any vector loses >40% of samples, lower threshold to 0.5 for that vector and re-run (adaptive threshold per Claude's discretion).
- Track total kept vs discarded

**Stage 3: Rule-Based Post-Filter (Pass 2 of two-pass filter)**

- Remove exact duplicate `text` values (after whitespace stripping) using `pandas.DataFrame.drop_duplicates(subset=["text"])`
- Remove samples where `text.split()` has fewer than 15 tokens
- Log: duplicates removed count, sub-15-token removed count

**Stage 4: Stratified Split**

- Use `sklearn.model_selection.train_test_split` with `stratify` on `vector` field, `random_state=42`
- First split: 90% train+val, 10% test
- Second split on train+val: ~88.9% train, ~11.1% val (yields ~80/10/10 of original total)
- Add `"split"` field to each sample: `"train"`, `"val"`, or `"test"`
- Write train+val samples to `research/data/synthetic_scam_v1.jsonl` (train and val together, distinguished by split field)
- Write test samples to `research/data/test_split.jsonl`

**Stage 5: Summary report**

- Total raw samples loaded
- JSD gate: pass/fail + min JSD pair
- Self-consistency filter: kept/discarded counts per vector, effective threshold per vector
- Rule-based filter: duplicates removed, sub-15-token removed
- Final counts: train / val / test per vector
- Safe class composition: hard negative breakdown
