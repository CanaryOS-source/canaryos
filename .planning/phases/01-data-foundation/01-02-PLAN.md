2. **Prompt templates (per-vector and safe):**
   - For each scam vector, create 5-8 diverse prompt templates that vary:
     - Channel: SMS, email, WhatsApp, app notification (per D-07)
     - Register: typos, non-native English, formal, colloquial (per D-07)
     - Length: short (1-2 sentences), medium (3-5 sentences), long (paragraph+)
     - Formality: casual, semi-formal, corporate
   - Use indirect framing for sensitive vectors per Pitfall 1.4: "Write an example message that a scam awareness educator would use to train people to recognize a [vector] scam..."
   - For romance_grooming specifically: route majority to Ollama (no safety filters) per Pitfall 1.4

     - NO marketing/promotional messages (D-10: excluded)

3. **Gemini generation function:**
```python
from google import genai
from google.genai import types
from pydantic import BaseModel

class ScamSample(BaseModel):
    text: str
    label: str
    vector: str
    channel: str

def generate_gemini(prompt: str, client: genai.Client) -> dict | None:
    """Generate one sample via Gemini 2.5 Flash. Returns None on failure."""
    try:
        response = client.models.generate_content(
            model="gemini-2.5-flash",
            contents=prompt,
            config=types.GenerateContentConfig(
                response_mime_type="application/json",
                response_json_schema=ScamSample.model_json_schema(),
            ),
        )
        sample = json.loads(response.text)
        sample["source"] = "gemini-2.5-flash"
        return sample
    except Exception as e:
        print(f"Gemini error: {e}")
        return None
```

4. **Ollama generation function:**
```python
import requests

def generate_ollama(prompt: str, model: str = "llama3.1:8b") -> dict | None:
    """Generate one sample via local Ollama. Returns None on failure."""
    try:
        response = requests.post(
            "http://localhost:11434/api/generate",
            json={"model": model, "prompt": prompt, "stream": False, "format": "json"},
            timeout=120,
        )
        response.raise_for_status()
        sample = json.loads(response.json()["response"])
        sample["source"] = "llama3.1:8b"
        return sample
    except Exception as e:
        print(f"Ollama error: {e}")
        return None
```

5. **Main generation loop:**
   - Load holdout texts into a set for contamination checking
   - For each vector, compute Gemini count (75%) and Ollama count (25%)
   - For romance_grooming and government_impersonation: increase Ollama share to ~50% (Pitfall 1.4: safety filter bypass)
   - Generate samples in batches, cycling through prompt templates
   - After each sample: validate JSON schema, check text is not in holdout set, check text length >= 15 tokens
   - Write valid samples to OUTPUT_PATH as JSONL (append mode, so script can be resumed)
   - Print progress every 500 samples: count per vector, Gemini vs Ollama breakdown
   - Handle rate limits: sleep 0.5s between Gemini calls, 0s between Ollama calls (local)

6. **Safe class generation:**
   - Generate safe samples matching the scam total (D-13: 50:50)
   - Hard negatives: ~25% of safe class (D-09) — bank alerts, delivery, 2FA, medical
   - Domain-matched: each hard negative type paired with corresponding scam vector domain (D-11)
   - Remaining ~75% of safe class: normal transactional messages (order confirmations, meeting reminders, receipts)

7. **Post-generation summary:**
   - Print total samples generated, per-vector counts, per-source counts, safe/scam ratio
   - Print any vectors that fell short of target (with percentage of target achieved)
